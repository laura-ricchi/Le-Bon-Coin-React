## Running the project

- Clone this repository :

```
Git Clone => https://github.com/laura-ricchi/Le-Bon-Coin-React
and  => cd leboncoin-projectlauraricchi
```

- Install packages :

```
npm install
```

- When the installation is complete :

```
bash
npm start
```
